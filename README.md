## Getting Started

1. Clone the repository
```bash
git clone https://github.com/sabir/chw-sabir.dev.git
```

2. Install dependencies
```bash
cd chw-sabir.dev
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```
